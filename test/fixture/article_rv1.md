## UI Shortcuts

The following keybindings work anywhere in Naughty's interface.

  * `Ctrl+Alt+S` save
  * `Ctrl+Alt+D` delete (pops up confirmation box)
  * `Ctrl+Alt+V` pretty print
  * `Ctrl+Alt+C` new page
